# Trifecta

Trifecta is a cutting-edge mobile application built with **Flutter** for the frontend and **TypeScript** for the backend API services. This app leverages the **Tuya Cloud API** to seamlessly interact with smart locks, providing users with a powerful tool to manage access to their properties.

## üöÄ Features

- **User Registration & Authentication**: Secure user registration and login system.
- **Device Pairing**: Users can easily pair their smart lock devices using a QR code.
- **Temporary Password Management**: 
  - Create temporary passwords for specific time frames.
  - View all generated temporary passwords in one place.
- **Profile Management**: A dedicated profile screen where users can view and update their information.
- **State Management**: Robust state management for seamless user experience and data consistency.
- **Attractive UI**: A user-friendly and aesthetically pleasing interface designed to enhance user interaction.

## ‚öôÔ∏è Setup & Installation

### Prerequisites

- Flutter SDK installed on your machine
- Node.js and npm installed
- Tuya Developer account for API access
- Firebase project set up with Authentication and Firestore

### Steps

1. **Clone the repository**:
    ```bash
    git clone https://github.com/your-username/trifecta.git
    cd trifecta
    ```

2. **Install dependencies**:
    - For Flutter:
      ```bash
      flutter pub get
      ```

3. **Set up Firebase**:
    - Add your Firebase configuration file to the Flutter project.
    - Set up Firebase Authentication and Firestore.

4. **Run the application**:
    - For Flutter:
      ```bash
      flutter run
      ```
